1. Scene One

一个 mr 里既有这次发版的内容，又有下次发版的内容，需要将下次发版的内容从 mr 中移除。
首先将不需要提交到 mr 的内容 git cherry-pick 到一个单独的分支，然后执行 git rebase 操作将不在本次发版中的内容 drop 掉，之后再重新提交分支
